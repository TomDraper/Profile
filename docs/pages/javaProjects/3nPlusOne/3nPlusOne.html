3n + 1 Problem

There is an unsolved math problem that is simple to understand but has eluded scientists for decades. It goes like this: If n is even you divide n by 2. If n is odd you times n by 3 and add 1. (3n + 1). The problem is that they always converge back to 1 eventually. Often but not always via square roots of two. The question is to prove whether n will always return to 1 or not.

I am not about to solve that problem.

I am however going to show you some cool ways to visualise the numbers that I made below. Largely inspired by the Numberphile YouTube channel, and in particular this video. The speed slider below will tell you how many times the sketch is iterating over the algorithm each frame.
