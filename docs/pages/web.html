Tools I regularly use:

P5 Editor is a quick to access browser based text editor with some common and easy-to-use libraries for vectors and shape drawing already installed, namely the p5 library.

Atom is my text-editor of choice and is what this website was coded with.

Glitch is a browser based text-editor with Node and server functionality preloaded. Truthfully I use it rarely but is necessary for anything beyond the simplest of websites.

Pendulums

A classic example of minor changes in initial conditions creating dramatic differences in outcomes. Set the start angle to 180. The difference is then the amount of degrees the pendulums can possibly spawn between (default is 1, so values between 179 and 181). Pendulums with more than 3 nodes (centre counts as a node) will have wildly different results. Damping and gravity are arbitrary.

Boids

Boids (first made by Craig Reynolds in 1987) are agents with rules that emulate realistic 'flocking' behaviour seen in the real world with animals such as birds or fish. I have included some parameters to create different results and display information. I'd stay away from the debug button on mobiles.
